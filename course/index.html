Phyloinformatics Course Notes

This course has five sessions:

  1. Introduction: Overview of course and introduction to mashups
  2. Geography: Making maps
  3. Taxonomy: Names and classifications
  4. Phylogeny: The search for trees
  5. Data mining: Extracting information from text and databases
  6. Project ideas

There is no course textbook, but you will find a lot of relevant material on http://iphylo.blogspot.com. There is also a collection of relevant papers on Mendeley.

Phyloinformatics Course is a group in Biological Sciences on Mendeley.
