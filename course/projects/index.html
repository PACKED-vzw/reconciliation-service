Projects

For some possible project ideas please the list below, or see Possible project ideas (these are intended for 4th year and/or Masters students).

Ideas

The ideas below are designed to give you some idea of the sort of projects that you could do. Feel free to suggest your own topic.

Data cleaning

  • Can you find a dataset that would be useful but is not easy to access? For example, maybe there is a list of animal traits which is in a PDF and you want to convert it to a file for further analysis (maybe even make it available to others to use through, say, figshare.

Data mining

  • Analyse the titles of papers describing new parasites, how could you extract information on their host associations?
  • There are tools to automatically find taxonomic names in text, e.g., http://gnrd.globalnames.org and http://www.ubio.org/tools/recognize.php. Test how effective they are by appling them to examples of scientific text. You could use, for example, articles in the journal Zookeys which has names already identified for you.

Visualisation

  • There are lots of ways to display taxonomic classifications, which do you think is most useful? How could you test this?

Taxonomy

  • There are an increasing number of tools to clean lists of taxonomic names, can you evaluate how useful these are?
  • How fast do taxonomic names change? What are the implications of this for people using these names? For example, for a public museum such as the Hunterian Zoology museum, how many names on the displays have changed since the labels were printed?

Mapping

  • Can you test the hypothesis that for a given taxon most of the new species are being found in protected areas?
  • Can you create some geophylogenies (using, say the tools on this site, or GenGIS) for a group of taxa? Can you use these tools to test a specific hypothesis of interest?
  • Can you write a regular expression to extract latitude and longitudes from a scientific paper? Can you use this to build a geographic database?
